  <action>
In BootScene.create(), after `this.generateNebula()` and before `this.scene.start('MainMenu')`, add a call to a new method `this.generateParticleTextures()`.

Add the new method to the class (after generateFallback):

```js
generateParticleTextures() {
  const g = this.make.graphics({ add: false });
  g.fillStyle(0xffffff, 1);
  g.fillCircle(2, 2, 2);
  g.generateTexture('particle', 4, 4);
  g.destroy();
}
```

Key requirements:
- Use `this.make.graphics({ add: false })` — NOT `this.add.graphics()` (avoids adding to scene display list)
- Fill with white `0xffffff` (tint at runtime controls final color)
- Circle center at (2, 2) with radius 2 within a 4×4 canvas
- Key is exactly `'particle'` (single quotes, lowercase)
- Call `g.destroy()` after generateTexture to free the temporary graphics object
