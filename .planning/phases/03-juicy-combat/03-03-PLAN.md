**Step 2: Replace showMuzzleFlash() body** — keep method name, replace entire body:

```js
showMuzzleFlash() {
  const tip = this.getTipPosition();
  const cfg = VFX.MUZZLE;
  const angleDeg = Phaser.Math.RadToDeg(this.sprite.rotation - Math.PI / 2);

  const emitter = this.scene.add.particles(tip.x, tip.y, 'particle', {
    speed: cfg.speed,
    lifespan: cfg.lifespan,
    scale: cfg.scale,
    tint: cfg.tint,
    angle: { min: angleDeg - cfg.angleSpread, max: angleDeg + cfg.angleSpread },
    maxParticles: cfg.count,
    emitting: false,
  });
  emitter.explode(cfg.count, tip.x, tip.y);
  emitter.on('complete', () => emitter.destroy());
}
```

Key requirements:
- `Phaser.Math.RadToDeg(this.sprite.rotation - Math.PI / 2)` converts sprite rotation to emitter angle (sprites point "up" at rotation 0; Phaser emitter angle 0 points right — subtract PI/2 to align)
- `angle: { min: angleDeg - cfg.angleSpread, max: angleDeg + cfg.angleSpread }` creates the ±30° forward cone
- `emitting: false` prevents auto-emit before explode() is called
- `emitter.on('complete', () => emitter.destroy())` — MANDATORY cleanup per D-03
- `this.scene.add.particles` — Turret uses `this.scene` to access the scene (not `this.add`)
