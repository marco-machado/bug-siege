**FILE 1: src/scenes/GameScene.js**

**1a. Add `shakeCamera(tier)` method** (add after the `playSfx` method, around line 334):

```javascript
shakeCamera(tier) {
  if (this.phase === 'gameover') return;
  const cfg = VFX.SHAKE[tier];
  if (!cfg) return;
  this.cameras.main.shake(cfg.duration, cfg.intensity, true);
}
```

- `force: true` (third parameter) implements D-08 "latest wins" -- each new shake interrupts the current one
- Gameover guard prevents shakes during scene transition (RESEARCH.md Pitfall 5)


Damage thresholds:
