**1. Replace `drawLightningChain(targets)` (currently lines 181-194):**

Replace the entire method with:
```javascript
drawLightningChain(targets) {
  const g = this.scene.add.graphics();
  const tip = this.getTipPosition();
  const cfg = VFX.ZAPPER_TRAIL;

  g.lineStyle(cfg.outerLineWidth, cfg.outerColor, cfg.outerAlpha);
  g.beginPath();
  g.moveTo(tip.x, tip.y);
  for (const t of targets) g.lineTo(t.x, t.y);
  g.strokePath();

  g.lineStyle(cfg.innerLineWidth, cfg.innerColor, cfg.innerAlpha);
  g.beginPath();
  g.moveTo(tip.x, tip.y);
  for (const t of targets) g.lineTo(t.x, t.y);
  g.strokePath();

  this.scene.time.delayedCall(cfg.lineDuration, () => g.destroy());

  this.spawnTrailParticles(tip, targets);
}
```

This draws two strokes:
- Outer: 6px wide, 0x9966ff purple at 0.4 alpha (soft glow)
- Inner: 2px wide, 0xeef2ff white at 1.0 alpha (bright core)
Creates the "hot center fading to cool edges" effect per D-04 and D-05.

**2. Add new `spawnTrailParticles(tip, targets)` method after `drawLightningChain`:**

```javascript
spawnTrailParticles(tip, targets) {
  const cfg = VFX.ZAPPER_TRAIL;
  const emitter = this.scene.add.particles(0, 0, 'particle-glow', {
    speed: { min: 5, max: 20 },
    lifespan: cfg.trailLifespan,
    scale: cfg.trailScale,
    tint: cfg.trailTint,
    alpha: cfg.trailAlpha,
    emitting: false,
  });

  const points = [tip, ...targets];
  for (let i = 0; i < points.length - 1; i++) {
    const a = points[i];
    const b = points[i + 1];
    const steps = cfg.particlesPerSegment;
    for (let s = 0; s <= steps; s++) {
      const t = s / steps;
      const x = a.x + (b.x - a.x) * t;
      const y = a.y + (b.y - a.y) * t;
      emitter.emitParticleAt(x, y, 1);
    }
  }

  this.scene.time.delayedCall(cfg.trailLifespan + 50, () => emitter.destroy());
}
```

Key details:
- Uses 'particle-glow' texture (8px soft circle from Plan 01) for the trail
- `emitting: false` -- particles are manually placed via `emitParticleAt()` at interpolated positions along each chain segment
- 4 particles per segment (cfg.particlesPerSegment) evenly spaced via lerp
- Cleanup uses `scene.time.delayedCall(lifespan + 50ms)` -- NOT `emitter.on('complete', ...)` because the 'complete' event may not fire on an emitter that was never formally started (only manual emitParticleAt calls). This is RESEARCH.md Pitfall 6.
- Trail lingers 100ms after the line fades (300ms trail - 200ms line = 100ms residual), creating the "residual energy dissipating" feel per D-05
