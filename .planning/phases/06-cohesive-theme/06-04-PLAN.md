**Step 4 — Carve-outs (do NOT migrate):**

- **Line 118:** `backgroundColor: '#000000aa'` — this is an 8-digit CSS RGBA string (6 hex digits + 2 alpha digits). The D-07 grep uses an anchored 6-digit pattern `#[0-9a-fA-F]{6}` so this literal will not match the regex (the alpha bytes extend it past 6 chars). It is a debug-overlay-only semi-transparent backdrop. Leave unchanged per 06-PATTERNS.md lines 69-70.

**Step 5 — Collapse rationale (D-06):**

- `#00ff88` (start-wave button, line 54) and `#88ff88` (debug text, line 117) → both express "active/positive accent" → both map to `ui.accentPrimary`. Debug text being accentPrimary is an intentional collapse (D-06 "collapse when intent is identical"); it slightly dims the neon-green debug readout toward the cosmic family.
- `#ffdd00` (credits, line 25) → `ui.warning`: credits is a "caution / keep an eye on your spending" semantic, which matches warning amber. Distinct from `ui.success` (healthy HP).
- `#aaaaaa` (HP label, line 39) → `ui.textMuted`: dim supporting label.
