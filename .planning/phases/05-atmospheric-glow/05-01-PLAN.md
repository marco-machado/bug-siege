  VIGNETTE: Object.freeze({
    x: 0.5,
    y: 0.5,
    radius: 0.5,
    buildStrength: 0.25,
    waveStrength: 0.40,

});

Notes for the executor:
- Color values MUST be numeric hex (`0xeef2ff`), NOT CSS strings — this matches the VFX convention (see `VFX.DEATH.swarmer.tint: 0x44ff44` in the existing file).

- NO `walls` entry under GLOW — walls never receive glow (per D-02). Turret constructor (plan 02) will branch on `type === 'wall'` to skip.
- `padding` field is included per RESEARCH Pitfall 1 (mandatory for small sprites with `outerStrength > 0`); RESEARCH Example 7 is authoritative.
- `transitionEase` is the string `'Sine.easeInOut'` — Phaser accepts string ease names (verified: existing `renderCore()` tween at GameScene.js:120 uses the same string form).
- Every nested object MUST be `Object.freeze(...)`-wrapped — project convention from VFX. Missing a freeze call will not throw at runtime, so the review is visual.
