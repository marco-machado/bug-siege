# Milestone v1.0: Cosmic Polish & Atmosphere

**Status:** ✅ SHIPPED 2026-04-20
**Phases:** 1-6
**Total Plans:** 21

## Overview

Transform the visual/audio identity of Bug Siege from utilitarian sci-fi to a cosmic nebula aesthetic with full juice and atmosphere. Deliver a satisfying, juicy tower defense that feels as good to watch as it is to play — every hit, kill, and wave clear delivers visual and audio impact within a cohesive cosmic atmosphere.

Scope was locked to visual/atmospheric polish; **AUDIO-0x requirements were dropped entirely** at close and are not carried forward.

## Milestone Summary

**Key Decisions:**

- **"Void Ethereal" palette** — Deep purples/blues foundation locked in `THEME` config for consistent cosmic atmospheric coloring across all render paths
- **Procedural animation over spritesheets** — Code-driven sin-wave squash/stretch, breathing, and alpha pulses avoided multi-frame asset creation
- **Procedural nebula background** — HTML5 Canvas gradient generation in BootScene, no external image assets
- **Dual-format THEME.ui** — Every UI semantic color exposed as both `hex` string and `num` numeric to match Phaser's mixed API (`setFillStyle(0x...)` vs `backgroundColor: '#...'`)
- **Semantic ceiling of 14 UI keys** — Constrained palette footprint to prevent drift
- **Glow via preFX, not postFX** — Per-sprite glow on turrets/core; vignette via camera postFX
- **Upgrade glow swap via property mutation** — `Glow.color = 0x...` instead of `preFX.clear() + addGlow()`; zero allocation
- **Phase 6 re-scope from Audio → Theme** — AUDIO-0x requirements dropped entirely; cohesive theme migration prioritized over ambient audio
- **Turret/core glow removed post-playtest** — Late commit `ed293a9` pulled glow halos; vignette and theme kept

**Issues Resolved:**

- Hardcoded color drift across 6+ source files (closed via THEME-03 migration)
- Orphaned tween state corruption on pooled Bug reuse (closed via `killTweensOf` in despawn)
- Slowfield static Graphics aura felt lifeless (replaced with particle pulse emitter)
- Core damage had no kinetic feedback (added tiered screen shake + shockwave ring)
- UI HUD shook with camera shake (moved to separate UIScene camera)
- Phaser mixed numeric/string color API pain (solved with dual-format `{ hex, num }` entries)

**Issues Deferred:**

- **AUDIO-01..04 (cosmic ambient drone, SFX pitch variation, concurrent sound limits, BGM crossfade)** — Dropped entirely at milestone close; not carried to next milestone
- **POLISH-03, POLISH-04, POLISH-05** (improved Game Over screens, round turret sprites, turret target-locking fix) — Deferred to v2 per REQUIREMENTS.md

**Technical Debt Incurred:**

- Performance metrics (frame rate, bug count, bullet count, particle count, memory) never empirically measured — STATE.md Performance Metrics table shows "Unknown" across the board
- Visual smoke-test (Gate C) is irreducibly manual — no automated visual regression harness
- Phase 6 re-scope was documented inline but ROADMAP.md "Ethereal Audio" phase was overwritten rather than archived as cancelled
- Late glow-removal commit (`ed293a9`) didn't update Phase 5 VERIFICATION artifacts to reflect the reversal
